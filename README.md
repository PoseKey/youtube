# Option Page for YouTube Motion Controller

### 1. Description
``` bash
!!Beware that this program is not a standalone wepage.
!!이 프로그램은 단독으로 사용되기 위한 프로그램이 아니며

This program is a module for YouTube Motion Controller.
이 프로그램은 YouTube Motion Controller의 한 모듈입니다.

The program is built on Vue.js
Vue.js 기반으로 제작되었으며

The program has 2 distinct features
2가지의 기능을 제공합니다.

Firstly, The program your poses in realtime.
첫번째 기능은 거울기능으로 자신을 볼 수 있습니다.

Secondly, The program has descriptions about the poses which is available.
두번째 기능은 미리 학습된 동작들에 대한 모습을 보여줍니다.
```

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
